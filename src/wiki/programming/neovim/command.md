# 覚えるコマンド集

- `ciw` カーソル上の単語を置き換え
- `ci{` `{}`内を置き換え
- `ctrl+r` `u`の逆操作
- `f <something>` 今いる行で`<something>`が最初に出てくる行に移動
- `shift+f <something>` fの戻る版
- `ctrl+o` 直前のカーソル位置に移動
- `gd` 定義に移動
- `r` 一文字置き換え
